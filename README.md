# Execução do programa

## Backend Spring

- Entre a pasta nozama back e execute o arquivo "scr > main > java > com > example > nozama > NozamaApplication.java"

## Frontend ReactJS

- Entre na pasta nozama-front e execute os seguintes comandos

```JavaScript
  npm install
```

```JavaScript
  npm start
```
